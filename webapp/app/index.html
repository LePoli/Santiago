Sé parte de la experiencia

NodeSchool es una iniciativa global llevada a cabo por voluntarios para la comunidad con dos objetivos:

  • Crear planes de estudio de programación de alta calidad.
  • Organizar eventos de aprendizaje comunitario.

NodeSchool Santiago está organizado por Javascript Chile.

Agenda

javascripting

Aprende lo básico de JavaScript. No necesitas experiencia previa programando.

learnyounode

Aprende lo básico de Node.js: asynchronous i/0, http.

stream-adventure

Aprende a construir streaming interfaces con .pipe()

ExpressWorks

Aprende lo básico del framework Express.js

Otros workshops disponibles para usuarios más avanzados*

¿Qué necesito para participar?

  • Un notebook con Node.js v4 or v5 instalado. Descárgalo desde el sitio oficial de nodejs.org.
  • Tu editor de texto favorito. Sublime Text, Atom, Vim, Emacs, Visual Studio Code, Notepad++, ...
  • Una terminal de tu preferencia. Todos los S.O. vienen con una terminal propia.
  • Una cuenta en Github (opcional).
